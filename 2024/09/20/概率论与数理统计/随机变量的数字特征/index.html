随机变量的数字特征

发表于2024-09-20|更新于2024-09-20|Math

1. 数学期望

1.1 数学期望的概念

离散型：设离散型随机变量$X$的分布律为$P(X=xi)=p_i,i=1,2,\cdots,$若级数$\displaystyle\sum{i=1}^{\infty}xip_i$绝对收敛，即$\displaystyle\sum{i=1}^{\infty}|x_i|p_i<+\infty$，则称

为离散型随机变量$X$的数学期望，也叫均值，记作$E(X)$

连续型：设$X$为连续型随机变量，其概率密度为$f(x)$，若$\displaystyle\int{-\infty}^{\infty}xf(x)\text{d}x$绝对收敛，即$\displaystyle\int{-\infty}^{\infty}|x|f(x)\text{d}x<\infty$，则称

为连续型随机变量$X$的数学期望。

对于级数$\displaystyle\sum_{i=1}^{\infty}x_ip_i$，若不绝对收敛则与$X$取值排列顺序有关，此时数学期望不存在。$E(X)$作为刻画$X$的某种特性的数值，有其客观意义而不受$X$取值的人为排序影响。连续型情况与离散型类似。

数学期望$E(X)$为一个确定的数，这个数反映了随机变量$X$的平均值。若把$X$的概率分布看作总质量为单位1的质点系，则$E(X)$可看成质点系的质心坐标。

1.3 数学期望的性质

  1. 设$X$为任意随机变量，则$X$的数学期望存在的充要条件为$E(|X|)<\infty$
  2. 设$X,Y$为任意两个数学期望存在的随机变量，且$X\leqslant Y$，则$E(X)\leqslant E(Y)$
  3. 设$C$为常数，则$E(C)=C$；设$X,Y$为任意两个数学期望存在的随机变量，$C$为任意常数，则$E(CX)=CE(X)$；设$X,Y$为任意两个数学期望存在的随机变量，则$X+Y$的数学期望也存在，且$E(X+Y)=E(X)+E(Y)$；设$X,Y$为任意两个数学期望存在的随机变量且$X$与$Y$相互独立，则$XY$的数学期望也存在，且$E(XY)=E(X)E(Y)$

2. 方差

2.1 方差的概念

设$X$为随机变量，若$E\left{[X-E(X)]^2\right}$存在，则称其为$X$的方差，记作$D(X)$或$Var(X)$，即

称$\sigma_X=\sqrt{D(X)}$为$X$的均方差或标准差。

方差的计算：

  1. 对于离散型随机变量，$D(X)=\displaystyle\sum_{i=1}^{\infty}[x_i-E(X)]^2p_i$
  2. 对于连续型随机变量，$D(X)=\displaystyle\int_{-\infty}^{\infty}[x-E(x)]^2f(x)\text{d}x$
  3. 用公式计算，$D(X)=E(X^2)-[E(X)]^2$

2.2 方差的性质

  1. 设$X$为任意的随机变量，则$X$的方差存在的充要条件为$E(X^2)<\infty$
  2. 设$X,Y$为任意的随机变量，若$X,Y$的方差都存在，则$X\pm Y$的方差也存在
  3. 设$C$为常数，则$D(C)=0$
  4. 设$C$为常数，$X$为方差存在的随机变量，则$D(CX)=C^2D(X)$
  5. 设$X,Y$为相互独立的随机变量，且$X,Y$方差都存在，则$D(X+Y)=D(X)+D(Y)$
  6. 设$X$为一个方差存在的随机变量，则对任意实数$C$，有$D(X)\leqslant E[(X-C)^2]$
  7. 设$X$为一个随机变量，则$D(X)=0$的充要条件为存在一个常数$C$，使得$P(X=C)=1$，其中$C=E(X)$

3. 协方差和相关系数

3.1 协方差和相关系数的概念

若$E[(X-E(X))(Y-E(Y))]$存在，则称它为随机变量$X$和$Y$的协方差，记为$\text{cov}(X,Y)$，即

当$D(X)>0,D(Y)>0$时，称

为随机变量$X$和$Y$的相关系数。

协方差的计算：

  1. 当$(X,Y)$为离散型随机变量时，$\text{cov}(X,Y)=\displaystyle\sum{i=1}^{\infty}\sum{j=1}^{\infty}[xi-E(X)][y_j-E(Y)]p{ij}$
  2. 当$(X,Y)$为连续型随机变量时，$\text{cov}(X,Y)=\displaystyle\int{-\infty}^{\infty}\int{-\infty}^{\infty}[x-E(X)][y-E(Y)]f(x,y)\text{d}y\text{d}x$
  3. $\text{cov}(X,Y)=E(XY)-E(X)E(Y)$
  4. $D(X\pm Y)=D(X)+D(Y)\pm 2\text{cov}(X,Y)$

3.2 协方差和相关系数的性质

协方差的性质：

  1. 若随机变量$X,Y$的协方差存在，则$\text{cov}(X,Y)=\text{cov}(Y,X)$
  2. 若随机变量$X,Y$的协方差存在，$a,b$为任意常数，则$\text{cov}(aX,bY)=ab\text{cov}(X,Y)$
  3. 若随机变量$X,Y,Z$满足协方差$\text{cov}(X,Z)$与$\text{cov}(Y,Z)$都存在，则$\text{cov}(X+Y,Z)$也存在，且$\text{cov}(Y,Z)=\text{cov}(X,Z)+\text{cov}(Y,Z)$
  4. Cauchy-Schwarz（柯西-施瓦茨不等式）:设$(X,Y)$为二维随机变量，若$X,Y$的方差存在，则$X,Y$的协方差也存在，且$|\text{cov}(X,Y)|\leqslant \sqrt{D(X)}\sqrt{D(Y)}$。当$D(X)>0,D(Y)>0$时，上式等号成立条件为存在常数$t_0$使得$P(Y-E(Y)=t_0(X-E(X)))=1$，且$t_0=\pm\sqrt{\frac{D(Y)}{D(X)}}$

相关系数的性质：

  1. $|\rho_{xy}|\leqslant 1$
  2. $|\rho_{xy}|=1$的充要条件为$P(Y^=\pm X^)=1$，其中$X^$与$Y^$分别为$X$和$Y$的标准化随机变量，此时称$X,Y$完全相关，且取“+”时为完全正相关，取”-“时为完全负相关。

相关系数的解释如下：
$|\rho{XY}|$的大小一定程度反映了$X$与$Y$之间线性关系的强弱，即由$X$的线性函数$aX+b$估计$Y$所产生的均方误差的大小。设$g(a,b)=E[(aX+b-Y)^2]$，称$\displaystyle\min{a,b}g(a,b)$为$Y$关于$X$的线性均方误差。由

不相关等价命题如下：
设随机变量$X$与$Y$的方差都存在，且$D(X)>0$，$D(Y)>0$，则下列命题等价。

  1. $X$与$Y$不相关
  2. $\rho_{XY}=0$
  3. $\text{cov}(X,Y)=0$
  4. $E(XY)=E(X)E(Y)$
  5. $D(X\pm Y)=D(X)+D(Y)$
  6. $D(X+Y)=D(X-Y)$

4. 随机变量的高阶矩

随机变量的高阶矩：设$X,Y$为随机变量，

  1. 若$E(|X|^k)<\infty(k=1,2,\cdots)$，则称$E(X^k)$为$X$的$k$阶原点矩，$E\left{[X-E(X)]^k\right}$为$X$的$k$阶中心矩。
  2. 若$E(|X|^k|Y|^l)<\infty(k,l=1,2,\cdots)$，则称$E(X^kY^l)$为$X$和$Y$的$k+l$阶混合原点矩，$E\left{[X-E(X)]^k[Y-E(Y)]^l\right}$为$X$的$k+l$阶混合中心矩。
